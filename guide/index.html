LibreDTE es un proyecto que tiene por objetivo proveer Facturación Electrónica Libre para Chile.

Aquí podrás encontrar la biblioteca escrita en lenguaje PHP para la integración con el Servicio de Impuestos Internos (SII) asociada a los Documentos Tributarios Electrónicos (DTE).

Esta biblioteca se construye, originalmente, para ser usada en la Aplicación Web de LibreDTE Edición Comunidad. En esta biblioteca solo estará lo básico, sin interfaces de usuario, para que un desarrollador pueda construir su propia aplicación.

Si deseas una aplicación "llegar y usar" sin preocuparte por instalaciones o servidores, revisa la Aplicación Web de LibreDTE Edición Enterprise. Podrás registrarte y usar la plataforma web de facturación junto a otros módulos.
