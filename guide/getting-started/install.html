Instalación

La forma recomendada de instalar es mediante composer:

composer require libredte/libredte-lib-core

Requerimientos

  • PHP 8.2 o superior.
  • Extensiones de PHP:
    • curl.
    • json.
    • mbstring.
    • openssl.
    • soap.
    • xdebug (opcional, solo para testing ).
